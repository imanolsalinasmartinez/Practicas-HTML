HTML es un lenguaje de marcado de marcado de texto que sirve para presentar páginas web desde el punto de vista
del Front End; un Front End en español significa la parte final que un usuario de Internet verá. Una etiqueta (marca)
delimita cada uno de los elementos que componen un documento de HTML.
Las páginas web están formadas por 2 partes: el encabezado y el cuerpo de la página, la primera tiene la función de
mostrar configuraciones y el cuerpo es donde se mostrará en sí toda la página.
La etiqueta HTML es el inicio de una página web en cualquier versión pero existe una etiqueta que se llama !Doctype
HTML que inicia una página web con la versión 5.
